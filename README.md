# Geekland Remote

 for RPi + touchscreen display, written in Kivy

- Kodi remote
- Philips Hue remote

using phue by Nathanaël Lécaudé
	https://github.com/studioimaginaire/phue

using rgb_cie by Benjamin Knight
	https://github.com/benknight/hue-python-rgb-converter
